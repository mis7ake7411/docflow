# DocFlow Lite Frontend

Vue 3 frontend scaffold for DocFlow Lite.

## Stack
- Vue 3
- Vite
- TypeScript
- Vue Router
- Pinia
- Axios
- Vue Query
- Element Plus

## Commands
```bash
npm install
npm run dev
npm run build
```

## Env
Copy `.env.example` to `.env` and adjust backend API base URL if needed.
